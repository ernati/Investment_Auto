# bond_trading_demo.py 앱 문서

## 개요
`bond_trading_demo.py`는 한국투자증권 Open Trading API를 사용하여 장내채권 거래 기능을 시연하는 데모 애플리케이션입니다.
한국은행통안증권(KR103502GA34)을 대상으로 매수/매도/정보조회 기능을 제공합니다.

## 파일 위치
- **경로**: `Investment_Auto/Scripts/apps/bond_trading_demo.py`
- **유형**: 애플리케이션 (Application)

## 주요 기능

### 1. 실전투자/모의투자 모드 지원
- `--demo` 옵션으로 모의투자 모드 사용
- 옵션 없이 실행하면 실전투자 모드로 동작
- 실전투자 시 확인 프롬프트 제공

### 2. 다양한 채권 거래 작업
- **정보 조회(info)**: 채권 기본 정보 조회 (기본 작업)
- **매수(buy)**: 채권 매수 주문 실행
- **매도(sell)**: 채권 매도 주문 실행

### 3. 안전 장치
- 실전투자 모드 시 사용자 확인 요구
- 상세한 에러 처리 및 로깅
- 키보드 인터럽트 처리

### 4. 유연한 파라미터 설정
- 수량 및 가격 조정 가능
- 명령줄 옵션을 통한 다양한 설정

## 사용 채권 정보
- **채권코드**: KR103502GA34
- **채권명**: 한국은행통안증권 (국고01500-5003(20-2))

## 모의투자 제약사항
- **정보 조회**: ✅ 지원됨 (채권 현재가 조회 API 사용)
- **매수/매도 주문**: ❌ 지원되지 않음 ("모의투자에서는 해당업무가 제공되지 않습니다")

## 사용법

### 기본 사용법

#### 실전투자 모드 (기본)
```bash
cd Investment_Auto/Scripts/apps

# 채권 정보 조회 (기본 작업)
python bond_trading_demo.py

# 채권 정보 조회 (명시적)
python bond_trading_demo.py --action info

# 채권 매수 (1구좌, 단가 10000)
python bond_trading_demo.py --action buy

# 채권 매도 (1구좌, 단가 10000)
python bond_trading_demo.py --action sell
```

#### 모의투자 모드 (`--demo` 옵션)
```bash
# 채권 정보 조회 (모의투자)
python bond_trading_demo.py --demo --action info

# 채권 매수 (모의투자, 2구좌, 단가 6700) - 실제로는 지원되지 않음
python bond_trading_demo.py --demo --action buy --quantity 2 --price 6700

# 채권 매도 (모의투자, 3구좌, 단가 6800) - 실제로는 지원되지 않음
python bond_trading_demo.py --demo --action sell --quantity 3 --price 6800
```

### 명령줄 옵션

#### 필수 옵션
- `--action {info|buy|sell}`: 실행할 작업 선택 (기본값: info)

#### 선택적 옵션
- `--demo`: 모의투자 모드 사용 (기본값: 실전투자)
- `--quantity INTEGER`: 주문수량 (기본값: 1)
- `--price STRING`: 주문단가 (기본값: "10000")

## 문제 해결 과정

### 2026-02-16 업데이트

#### 1. "없는 서비스 코드 입니다" 오류 해결
**문제 증상**: 
```
❌ 채권 정보 조회 실패: 없는 서비스 코드 입니다
```

**원인**: KIS 인증 모듈(`kis_auth.py`)에서 모든 'C'로 시작하는 TR_ID를 모의투자용으로 'V'로 변환하는 로직이 있었음. 채권 정보 조회용 TR_ID "CTPF1114R"이 "VTPF1114R"로 잘못 변환됨.

**해결 방법**: 
- 채권 정보 조회용 'CTPF'로 시작하는 TR_ID는 변환하지 않도록 수정
- `kis_auth.py` 수정: `tr_id.startswith("CTPF")` 조건 추가

#### 2. "모의투자 TR 이 아닙니다" 오류 해결
**문제 증상**:
```
❌ 채권 정보 조회 실패: 모의투자 TR 이 아닙니다
```

**원인**: 채권 기본 정보 조회 API(CTPF1114R)가 모의투자 환경에서 지원되지 않음.

**해결 방법**:
- 모의투자에서는 채권 현재가 조회 API(FHKBJ773400C0) 사용
- `kis_bond_trading.py`에 `get_bond_price()` 메서드 추가
- `get_bond_info()` 메서드에서 모의투자 시 자동으로 현재가 조회를 사용하도록 수정

#### 3. 최종 결과
✅ **정보 조회 성공**: 모의투자에서 채권 현재가 정보 조회 가능
❌ **매수/매도 제한**: 모의투자에서는 채권 거래 주문이 지원되지 않음

**응답 데이터 예시**:
```json
{
  "stnd_iscd": "KR103502GA34",
  "hts_kor_isnm": "국고01500-5003(20-2)", 
  "bond_prpr": "6708.00",
  "prdy_vrss_sign": "2",
  "bond_prdy_vrss": "38.00",
  "prdy_ctrt": "0.57",
  "acml_vol": "7491160",
  "ernn_rate": "3.595"
}
```

## API 정보

### 사용되는 TR_ID
- **채권 정보 조회 (실전투자)**: CTPF1114R
- **채권 현재가 조회 (모의투자)**: FHKBJ773400C0  
- **채권 매수 주문**: TTTC0952U (실전투자) / VTTC0952U (모의투자)
- **채권 매도 주문**: TTTC0958U (실전투자) / VTTC0958U (모의투자)

### 지원 환경
| 기능 | 실전투자 | 모의투자 |
|------|----------|----------|
| 정보 조회 | ✅ 지원 | ✅ 지원 (현재가 조회) |
| 매수 주문 | ✅ 지원 | ❌ 미지원 |
| 매도 주문 | ✅ 지원 | ❌ 미지원 |
